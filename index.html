Documentation projet cartiflette

Un exercice pour illustrer la simplicité d’usage des fonds de carte du projet cartiflette1

Note

Le projet s’enrichit progressivement de fonctionnalités, n’hésitez pas à consulter régulièrement cette page.

Pour le moment, cartiflette est fourni principalement sous forme de package Python . Quelques fonctionnalités sont portées en JavaScript avec le présent notebook.

Le développement d’une API pour offrir de la flexibilité dans la récupération des fichiers tout en laissant la liberté à l’utilisateur est prévue. N’hésitez pas à proposer de l’aide sur Github inseefrlab/cartiflette

Récupérer des données France entière

Les premières données qu’il est possible de récupérer avec l’outil cartiflette sont des données France entière.

Ce n’est pas là où le package apporte le plus de fonctionnalités novatrices mais cela peut être pratique pour récupérer facilement les contours administratifs (voir 👇 pour ceux disponibles).

Il existe deux approches pour cela:

  1. Utiliser la fonction get_vectorfile_ign qui va récupérer directement les données depuis le site de l’IGN. Comme il est nécessaire de télécharger les données France entière pour cela et que le format utilisé est le shapefile, les données sont volumineuses (de l’ordre de 500Mo). Cela peut donc demander un certain temps ;
  2. Utiliser l’une des fonctions de cartiflette. Ce cas est démontré ci-dessous
